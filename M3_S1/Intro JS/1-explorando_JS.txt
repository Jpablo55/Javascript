Parte 1: Explorando JavaScript


(1)
Javascript al ser un lenguaje de programación a diferencia de HTML y CSS, 
carga con la lógica del programa desarrollado, y le da la funcionalidad a los elementos de la página/app.

(2)
Características principales.
1. Es un lenguaje interpretado: Lo que significa que no necesita ser compilado.
 Los navegadores web como Google Chrome, Firefox, o Safari interpretan el código JavaScript al momento de la ejecución,
 lo que permite ver los cambios y resultados de inmediato. Esto es genial para principiantes, porque puedes experimentar y aprender al vuelo. 

2. Es un lenguaje basado en eventos (Event-Driven): La mayor parte de lo que haces con JavaScript en la web gira en torno a responder a eventos del usuario,
 como clics, movimiento del ratón o cambios en un formulario. 

3. Es flexible y dinámico: A diferencia de lenguajes más estrictos, JavaScript te permite escribir código sin preocuparte demasiado por ciertos detalles
 (como declarar tipos de variables, algo obligatorio en otros lenguajes). Esto lo hace más accesible para aprender,
  pero también significa que el programador debe tener cuidado para evitar errores. 

4. Interactividad: permite cambiar el contenido de una página web estática sin recargarla por completo como respuesta a un evento realizado por el usuario. 

5. Paradigma de programación orientado a objetos (OOP): JavaScript admite el paradigma de Programación Orientada a Objetos (Object-Oriented Programming),
 lo que significa que puedes estructurar tu código en objetos, que son entidades que agrupan datos y comportamientos. 

6. Multiplataforma: funciona en todos los navegadores modernos, por lo que es universal en el desarrollo web.
